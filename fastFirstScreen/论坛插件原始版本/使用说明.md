# 作用

首屏启动优化

# 使用方式

将首屏图片命名为`first.jpg`放入`settings`根目录,在Cocos中构建即可

# 支持平台

理论上微信、抖音、快手小游戏都支持，目前只测试微信

# 支持版本

2.4.10上测试，理论支持2.4.x，不支持其他版本

# 售后

代码开源，不负责任何售后，构建中出错，请卸载该插件，插件没有开关选项，推荐将插件安装在项目。
